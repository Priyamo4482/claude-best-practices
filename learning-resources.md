## Where to Start If Overwhelmed
Don't try to consume all of this at once.

- **Week 1** -> Read Lilian Weng guide + ReAct paper. Watch Karpathy's LLM intro.
- **Week 2** -> Take the LangGraph course on DeepLearning.AI. Clone the LangChain repo.
- **Week 3** -> Read Anthropic's agent guide + Skills guide. Start building something small.
- **Week 4** -> Go deeper with the papers. Pick one newsletter and stay consistent.
