How to contribute to the repo (no terminal needed)

1. Fork it
Go to https://github.com/AravindKurapati/claude-best-practices -> click Fork (top right) -> Create fork
2. Find the right file
Open 00-index.md  it tells you which file your tip belongs in. When in doubt, it probably goes in one of these:

New Claude Code trick -> agentic-workflow-best-practices.md
MCP server or tool -> mcp-tools.md
Skill authoring tip -> skills-best-practices.md
Learning resource -> learning-resources.md

3. Edit the file
Open the file in your fork -> click the pencil icon -> add your content. Match the style of what's already there: short header, bullet points, no fluff.
4. Update CHANGELOG.md
Add a line at the top under ## [Unreleased]:
### Added
- [one line description of what you added and which file]
5. Update 00-index.md (only if needed)
If you added a whole new section that someone might need to find, add it to the index. If you just added a tip inside an existing section, skip this.
6. Open a PR
In your fork -> click Contribute -> Open pull request -> write what you added and where you found it -> Create pull request

What makes a good tip:

"Do X when Y"  actionable, not just informational
You've actually tried it, or it's from a credible source
Credit the original author if it came from somewhere
